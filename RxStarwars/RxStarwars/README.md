# RxStarwars

[Starwars 등장인물 API](https://swapi.dev/)를 받아 UITableView로 나타내는 토이프로젝트 입니다.

### 달성조건
1. RxSwift 사용 (SPM 사용)
2. MVVM
3. UITableViewDelegfate, UITableViewDatasource 대신 RxCocoa 활용.
4. Alamofire 등 리퀘스트 관련 외부 프레임워크 사용 🙅‍♂️

### API
- [Starwars 등장인물 API](https://swapi.dev/)
